# 📊 Marketing Campaign Performance Analysis (SQL Project)
## 📌 Overview
This project analyzes a multi-channel marketing campaign dataset to uncover insights on performance, efficiency, and return on investment. 
________________________________________
## 📁 Dataset
The dataset contains 6 related tables, of which 5 are dimension tables, and 1 is a fact table.
- campaign.csv
- campaigns.csv
- channels.csv
- devices.csv
- performance.csv
- regions.csv
________________________________________
## 🎯 Objectives
- Evaluate campaign performance
- Identify high-performing channels and regions
- Optimize marketing spend using data-driven insights
________________________________________
## 🛠 Tools Used
- MySQL
  - CTE
  - Joins
  - Aggregation
________________________________________
## 📊 Key Metrics

The following metrics were computed to evaluate campaign performance by device, channel, and region.
- CTR (Click-Through Rate)
- Conversion Rate
- CPC (Cost per Click)
- CPA (Cost per Acquisition)
- ROAS (Return on Ad Spend)

________________________________________
## 🔍 Key Insights
- Identified top-performing channels by ROAS
- Discovered regional performance differences
- Found inefficiencies in high-spend campaigns
________________________________________
## 💡 Recommendations
- Reallocate budget to high-ROI channels
- Optimize underperforming campaigns
- Focus on high-converting regions and devices
